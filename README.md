# Physics Simulation

A physics simulation project built with JavaScript/HTML5 Canvas.

## Description

This project demonstrates various physics concepts and simulations using web technologies.

## Getting Started

Open `index.html` in your web browser to run the simulation.

## Features

- Real-time physics simulation
- Interactive canvas-based visualization
- Customizable parameters

## Technologies Used

- HTML5 Canvas
- JavaScript
- CSS
